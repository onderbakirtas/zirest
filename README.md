 # Zirest
 
 Zirest is a simple REST API desktop client built with GTK (GJS) and TypeScript.
 
 ## Development
 
 This project is built with [Vite](https://vite.dev/) and [GTK](https://www.gtk.org/). You can find the development documentation in the [DEVELOPMENT.md](DEVELOPMENT.md) file.
